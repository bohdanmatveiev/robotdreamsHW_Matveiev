В першу чергу на віртуальній машині встановив nginx за допомогою команди 
"apt install nginx -y "
і перевірив його статус - screen1
Також перевірив справність запустивши nginx в браузері - screen2
Розгорнув простий додаток, а саме статичний сайт з метою подивитись як все пов'язано і чому воно функціонує - screen3, створений статичний сайт відкрився за посиланням, а отже Nginx успішно обслуговує статичні файли з кастомної директорії.
Також встановив та видалив ppa-репозиторій за допомогою команд: 
add-apt-repository ppa:ondrej/nginx -y
apt update
apt upgrade nginx
nginx -v (стала версія 1.28.0, була 1.24.0)
Після цього за допомогою ppa-purge повернувся до попередньої версії: 
apt install ppa-purge
ppa-purge ppa:ondrej/nginx
І версія nginx знову стала 1.24.0
